从v1和v2的学习我们可以看出，人们为了能够清晰的理清思路，所有在一个项目里有很多文件夹，
每个文件夹下有自己的功能比如model  view 或者control。要用到的时候对于java和python就是from...import ...

对于php就是require或者include.  

从上面这些我们不禁疑问？言下之意我们也可以把所以的东西房放到一个文件里然后把所以的东西都写入到一个文件中。
这样从理论上可以的但是很乱 很难修改。

所以我们才采用了三层模型 结果清晰。

所以说脚本和 一个程序的区别就是：
一个程序是很多脚本写出了的，不同的脚本实现不同的功能，如果要调用另外一个脚本的功能就要通过import进行导入。
这就是程序和单个脚本的区别。
